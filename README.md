# 📊 Excel Retail & Sales Data Analysis Project

This project was completed as part of a **Data Technician Bootcamp** and focuses on analysing retail and sales datasets using **Microsoft Excel**. The goal was to practice essential data analysis techniques such as data cleaning, aggregation, categorisation, and visualisation.

The project explores sales and retail datasets to uncover insights about **customer demographics, sales performance, and product demand** through formulas, PivotTables, and charts.

---

## 🚀 Project Overview

Using Excel, I analysed retail and product sales datasets to:

- Explore **customer demographics and purchasing patterns**
- Summarise **sales performance across regions and products**
- Categorise sales volumes using conditional logic
- Create **data visualisations** to communicate insights

This project helped build practical spreadsheet skills commonly used in **data analysis and business intelligence roles**.

  - Filtered datasets to analyse specific attributes

### Excel Formulas & Functions

Used a variety of built-in Excel functions including:

- `SUM` – Calculate total commissions and sales values  
- `SUMIF` – Conditional aggregation of sales data  
- `AVERAGE` – Calculate average commission values  
- `AVERAGEIF` – Average values based on conditions  
- `DATE` – Create valid date values  
- `MONTH` – Extract month from date values  
- `YEAR` – Extract year from date values  
- `UNIQUE` – Identify unique values within datasets  
- `VLOOKUP` – Retrieve related information across tables  
- `SWITCH` – Categorise products based on sales volume  

Example logic used:

    =SWITCH(TRUE, C2 > 600, "High", C2 >= 300, "Medium", "Low")

This formula categorises sales performance into **High, Medium, or Low** tiers.
